How it works

Set it up once.
Never think about it again.

Five steps run every day at 8am, 1pm, and 6pm. The bot handles everything.

Step 01
💡
Finds trending content
Monitors Reddit, Hacker News, TechCrunch, The Verge, Wired, and VentureBeat for the most engaging stories.
1
Step 02
✏️
Writes the caption
Claude AI crafts a punchy, professional LinkedIn post — sharp tone, real facts, ends with something people comment on.
2
Step 03
📊
Scores for virality
Every post scored 1–10 on hook strength, emotional trigger, and comment potential. Below 7/10 is skipped.
3
Step 04
📷
Finds the image
Claude Vision picks the most relevant image from Wikipedia, og:images, and Bing. Rejects anything generic.
4
Step 05
🚀
Posts to LinkedIn
Published via LinkedIn API. No browser. No manual steps. Just posts — every single day, automatically.
5
Features

Everything you need.
Nothing you don't.

Built lean. Every part of the pipeline does exactly one job.

💡
Multi-source pipeline
Pulls from Reddit's top AI/tech subreddits, Hacker News, and 8 professional RSS feeds simultaneously. Always fresh.
🤖
Claude-powered writing
Not generic AI slop. Real facts, natural tone, sounds like a sharp professional — not a LinkedIn guru.
🎯
Viral score filter
Scored on hook strength, emotional trigger, comment potential. Below 7/10 doesn't post. Ever.
📷
Smart image selection
Wikipedia, og:images, Bing — Claude Vision picks the winner and rejects anything generic or off-topic.
🔒
Duplicate prevention
7-day history with fuzzy matching. Never posts the same story twice — even if it's been reworded.
⚡
Fully automated
Posts at 8am, 1pm, 6pm daily. Set once, runs forever in the background. Zero daily effort required.

Get started

Running in under 5 minutes.

Clone, configure, run. That's it.

01
Clone the repo
git clone from GitHub and cd into the autopost folder.
02
Install dependencies
pip install -r requirements.txt — everything in one command.
03
Add your API keys
Claude API key, LinkedIn token — copy .env.example and fill it in.
04
Run it and walk away
python main.py — configure your niche in settings.py, then let it run forever.
quickstart.sh
# 1. Clone the repo
git clone github.com/Saifnjit/autopost
cd autopost

# 2. Install dependencies
pip install -r requirements.txt

# 3. Add your API keys
cp .env.example .env
# Fill in ANTHROPIC_API_KEY, LINKEDIN_TOKEN...

# 4. Run it
python main.py

# You're live. Walk away.
